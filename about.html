🚀 Sobre mí

Soy Sheynner, estudiante de desarrollo web apasionado por la tecnología y la creación de experiencias digitales intuitivas. Me encanta aprender constantemente y aplicar nuevos conocimientos en proyectos reales que desafíen mi creatividad y habilidades profesionales.

💻 Frontend

Manejo tecnologías como HTML, CSS y JavaScript para construir sitios web atractivos y funcionales. Me especializo en diseño responsive, asegurando que cada proyecto se vea bien en cualquier dispositivo. También estoy explorando frameworks como React para mejorar la interactividad y el rendimiento.

⚙️ Backend

Actualmente estoy aprendiendo Node.js para desarrollar servidores eficientes y escalables. También estoy profundizando en el uso de bases de datos como MongoDB y MySQL, con el objetivo de construir aplicaciones completas que conecten el frontend con el backend de forma segura y dinámica.

🛠️ Herramientas

Utilizo herramientas como Git y GitHub para el control de versiones y la colaboración en proyectos. Mi entorno de desarrollo favorito es Visual Studio Code, y también tengo experiencia desplegando sitios web mediante GitHub Pages. Estas herramientas me permiten trabajar de forma organizada y profesional.
